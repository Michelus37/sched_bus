# Card templates

This directory can hold rank and suit templates for generic playing cards.

- `ranks/`  – small templates for card ranks like `a.png`, `k.png`, `10.png`, `q.png`, etc.
- `suits/`  – small templates for card suits like `hearts.png`, `spades.png`, `clubs.png`, `diamonds.png`.

The detector can also load full-card templates from `templates/cards/*.png` if you want to add them later.
